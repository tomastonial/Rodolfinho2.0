## Sinopse do Jogo

**Cão Corredor** é um jogo desenvolvido em Python utilizando a biblioteca Pygame. O jogador controla um cachorro que permanece parado na rua enquanto o cenário e os obstáculos se movimentam em sua direção, criando a sensação de uma corrida constante pela cidade.

Para sobreviver, o jogador deve movimentar o cachorro para cima e para baixo, desviando dos carros que aparecem pelo caminho. Além disso, é preciso realizar pausas estratégicas para manter o bem-estar do personagem e continuar acumulando pontos.

Com uma estética em pixel art e mecânicas simples de aprender, o jogo desafia os reflexos e a atenção do jogador, que precisa reagir rapidamente aos obstáculos enquanto busca alcançar a maior pontuação possível.

## Objetivo

Sobreviver pelo maior tempo possível, desviando dos carros, realizando pausas quando necessário e acumulando pontos para entrar no ranking dos melhores jogadores.


## Como Executar

1. Instale o Python 3.
2. Instale as dependências necessárias:

   ```bash
   pip install pygame pyttsx3
   ```
3. Execute o arquivo principal:

   ```bash
   python main.py
   ```
